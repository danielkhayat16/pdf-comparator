# PDF Comparator

A simple tool to compare two PDF documents and highlight the differences.

## Features

- Upload two PDFs
- Extract text
- Compute differences
- Highlight changes

## Tech Stack

Backend:
- Python
- FastAPI
- pdfminer / pymupdf

Frontend:
- HTML
- JavaScript
- Tailwind (optional)

## Use cases

- Legal document comparison
- Mainframe letter validation
- Contract verification
- Audit and compliance

## Roadmap

MVP:
- text extraction
- diff algorithm
- web interface

Future:
- visual diff
- OCR
- AI semantic diff
- batch comparison
